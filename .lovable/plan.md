## Design System

**Color Palette:**
- Deep stone black/charcoal backgrounds
- Desert sand and warm beige tones
- Rich gold accents for highlights
- Papyrus cream for text sections

**Typography:**
- Bold serif display font for headings (engraved ancient feel)
- Clean sans-serif for body text

**Visual Elements:**
- Hieroglyphic-inspired decorative patterns
- Sand particle effects
- Stone and papyrus textures
- Gold border frames and dividers

## Homepage Sections

### 1. Hero Section (Full-screen)
- Dramatic temple/pyramid background with warm golden lighting
- Floating sand particle animation
- **"EPOCH"** title with engraved stone effect
- Tagline: "Uncover the Legend"
- Two CTAs: "Register Now" (gold) + "Explore Events" (outline)
- Subtle scroll indicator

### 2. Pro Shows Section
- Heading styled as ancient royal decree
- 3-5 artist cards with gold-framed borders
- Each card: Artist image, name, date placeholder
- Hover animations with golden glow
- Placeholder content you can edit later

### 3. Events Section - "The Trials"
- Events presented as ancient quests/challenges
- Grid of 3 event cards: Dance, Music, Arts
- Each with themed icon, description, and prize info
- Stone tablet styling with hover reveal effects

### 4. Story/Lore Section
- Parchment/papyrus background texture
- Cinematic narrative about the fest's legend
- Scroll-triggered fade-in animations
- Decorative hieroglyphic borders

### 5. Schedule Timeline
- Styled as an ancient journey map
- Day-by-day visual timeline
- Milestone markers with event highlights
- Horizontal scroll on mobile

### 6. Past Highlights Gallery
- Image gallery with parallax effects
- Animated stats counters (attendees, events, artists)
- Stone frame styling

### 7. Sponsors Section
- Logos displayed on stone plaques
- Arranged in pyramid/temple column layout
- Placeholder slots for sponsor logos

### 8. Final CTA Section
- Epic closing message
- Large "Register Now" button
- Background with dramatic temple imagery
